Calculator Application

📌 Project Overview

This is a simple and responsive Calculator Application developed using HTML, CSS, and JavaScript. It performs basic arithmetic operations through an interactive and user-friendly interface.

🚀 Features

- Addition, subtraction, multiplication, and division
- Clear and delete functionality
- Responsive design for different screen sizes
- Easy-to-use interface
- Real-time calculation display

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

📂 Project Structure

Calculator/
│
├── index.html
├── style.css
└── script.js

▶️ How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open "index.html" in your browser.

🎯 Learning Outcomes

- Understanding JavaScript functions and events
- Implementing arithmetic operations
- Creating responsive user interfaces
- Improving front-end development skills
