Portfolio Website

📌 Project Overview

This is a personal Portfolio Website developed using HTML, CSS, and JavaScript. It showcases skills, projects, education, and contact information in a professional and user-friendly manner.

🚀 Features

- Responsive design for all devices
- Modern and attractive user interface
- About Me section
- Skills and Projects showcase
- Contact section with social links
- Smooth navigation and interactive elements

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

📂 Project Structure

Portfolio-Website/
│
├── index.html
├── style.css
├── script.js
└── images/

▶️ How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open "index.html" in your browser.

- Building responsive web pages
- Creating professional portfolio layouts
